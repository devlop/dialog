dialog

Drop-in replacement for the native dialog methods (alert, confirm, prompt).

View on GitHub

Examples

alert with title

// alert with title
await dialog.alert('Your changes have been saved.', {
    title: 'Saved',
});

alert with custom button text

// alert with title
await dialog.alert('Your changes have been saved.', {
    title: 'Saved',
    okText: 'Alright!',
});
